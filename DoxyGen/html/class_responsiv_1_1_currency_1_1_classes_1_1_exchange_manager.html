registerCallback ( callable  $callback)

Registers a callback function that defines a converter type. The callback function should register converters by calling the manager's registerConverters() function. The manager instance is passed to the callback function as an argument. Usage: 

  ExchangeManager::registerCallback(function($manager){
      $manager->registerConverters([...]);
  });
