Detailed Description

Stripe Update Transfer Reversal Request.

Updates the specified reversal by setting the values of the parameters passed. Any parameters not provided will be left unchanged.

This request only accepts metadata and description as arguments.

// Once the transaction has been authorized, we can capture it for final payment. $transaction = $gateway->updateTransfer(array( 'transferReference' => '{TRANSFER_ID}', 'reversalReference' => '{REVERSAL_ID}', 'metadata' => [], )); $response = $transaction->send();

https://stripe.com/docs/api#update_transfer_reversal
