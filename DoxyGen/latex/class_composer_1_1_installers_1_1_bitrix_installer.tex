\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_composer_1_1_installers_1_1_installer}{Installer}} for Bitrix Framework. Supported types of extensions\+:
\begin{DoxyItemize}
\item {\ttfamily bitrix-\/d7-\/module} — copy the module to directory {\ttfamily bitrix/modules/$<$vendor$>$.$<$name$>$}.
\item {\ttfamily bitrix-\/d7-\/component} — copy the component to directory {\ttfamily bitrix/components/$<$vendor$>$/$<$name$>$}.
\item {\ttfamily bitrix-\/d7-\/template} — copy the template to directory {\ttfamily bitrix/templates/$<$vendor$>$\+\_\+$<$name$>$}.
\end{DoxyItemize}

You can set custom path to directory with Bitrix kernel in {\ttfamily composer.\+json}\+:
