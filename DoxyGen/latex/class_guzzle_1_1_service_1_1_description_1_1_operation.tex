\item http\+Method\+: (string) HTTP method of the operation

\item response\+Notes\+: (string) Information about the response returned by the operation

\item error\+Responses\+: (array) Errors that could occur when executing the command. \item data\+: (array) Any extra data that might be used to help build or serialize the operation
\item additional\+Parameters\+: (null$\vert$array) \mbox{\hyperlink{class_guzzle_1_1_service_1_1_description_1_1_parameter}{Parameter}} schema to use when an option is passed to the operation that is not in the schema
