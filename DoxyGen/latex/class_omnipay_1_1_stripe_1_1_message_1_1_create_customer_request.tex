\doxysubsection{Detailed Description}
\mbox{\hyperlink{namespace_omnipay_1_1_stripe}{Stripe}} Create Customer Request.

Customer objects allow you to perform recurring charges and track multiple charges that are associated with the same customer. The API allows you to create, delete, and update your customers. You can retrieve individual customers as well as a list of all of your customers.\hypertarget{class_omnipay_1_1_stripe_1_1_message_1_1_create_customer_request_autotoc_md207}{}\doxyparagraph{Examples}\label{class_omnipay_1_1_stripe_1_1_message_1_1_create_customer_request_autotoc_md207}
