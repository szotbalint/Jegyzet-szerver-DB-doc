If you are having general issues with Omnipay, we suggest posting on \href{http://stackoverflow.com/}{\texttt{ Stack Overflow}}. Be sure to add the \href{http://stackoverflow.com/questions/tagged/omnipay}{\texttt{ omnipay tag}} so it can be easily found.

If you want to keep up to date with release anouncements, discuss ideas for the project, or ask more detailed questions, there is also a \href{https://groups.google.com/forum/\#!forum/omnipay}{\texttt{ mailing list}} which you can subscribe to.

If you believe you have found a bug, please report it using the \href{https://github.com/thephpleague/omnipay-common/issues}{\texttt{ Git\+Hub issue tracker}}, or better yet, fork the library and submit a pull request.
