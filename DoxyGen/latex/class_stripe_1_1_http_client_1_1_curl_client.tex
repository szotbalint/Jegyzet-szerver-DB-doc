\mbox{\hyperlink{class_stripe_1_1_http_client_1_1_curl_client}{Curl\+Client}} constructor.

Pass in a callable to \$default\+Options that returns an array of CURLOPT\+\_\+$\ast$ values to start off a request with, or an flat array with the same format used by curl\+\_\+setopt\+\_\+array() to provide a static set of options. Note that many options are overridden later in the request call, including timeouts, which can be set via \mbox{\hyperlink{class_stripe_1_1_http_client_1_1_curl_client_ad5c3315d293ca64fa60d69af54fc5de8}{set\+Timeout()}} and \mbox{\hyperlink{class_stripe_1_1_http_client_1_1_curl_client_a25b261a05a130a79f45fb762022de7df}{set\+Connect\+Timeout()}}.

Note that \mbox{\hyperlink{class_stripe_1_1_http_client_1_1_curl_client_ac931426d5baf2373053abfde3e08f3fe}{request()}} will silently ignore a non-\/callable, non-\/array \$default\+Options, and will throw an exception if \$default\+Options returns a non-\/array value.
