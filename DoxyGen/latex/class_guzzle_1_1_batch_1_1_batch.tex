\doxysubsection{Detailed Description}
Default batch implementation used to convert queued items into smaller chunks of batches using a \{\begin{DoxySeeAlso}{See also}
Batch\+Divisor\+Iterface\} and transfers each batch using a \{

\mbox{\hyperlink{interface_guzzle_1_1_batch_1_1_batch_transfer_interface}{Batch\+Transfer\+Interface}}\}.
\end{DoxySeeAlso}
Any exception encountered during a flush operation will throw a \{\begin{DoxySeeAlso}{See also}
Batch\+Transfer\+Exception\} object containing the batch that failed. After an exception is encountered, you can \mbox{\hyperlink{class_guzzle_1_1_batch_1_1_batch_a7751f77b5263bcf940ece6e824a05b38}{flush}} the batch again to attempt to finish transferring any previously created batches or queued items. 
\end{DoxySeeAlso}
