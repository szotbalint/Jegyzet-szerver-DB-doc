\doxysubsection{Detailed Description}
Redirect Response interface

This interface class defines the functionality of a response that is a redirect response. It extends the \mbox{\hyperlink{interface_omnipay_1_1_common_1_1_message_1_1_response_interface}{Response\+Interface}} interface class with some extra functions relating to the specifics of a redirect response from the gateway.
