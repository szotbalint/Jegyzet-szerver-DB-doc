\doxysubsection{Detailed Description}
Fetch Payment Methods Response interface

This interface class defines the functionality of a response that is a \char`\"{}fetch payment method\char`\"{} response. It extends the \mbox{\hyperlink{interface_omnipay_1_1_common_1_1_message_1_1_response_interface}{Response\+Interface}} interface class with some extra functions relating to the specifics of a response to fetch the payment method from the gateway. This happens when the gateway needs the customer to choose a payment method.
