Simple backend To-\/\+Do List plugin. It allows to create tasks list for each backend user.\hypertarget{md_renatio_todos__r_e_a_d_m_e_autotoc_md593}{}\doxysection{Features}\label{md_renatio_todos__r_e_a_d_m_e_autotoc_md593}

\begin{DoxyItemize}
\item Manage lists and tasks
\item Lists of tasks belongs to backend user
\item Task reminder email
\end{DoxyItemize}\hypertarget{md_renatio_todos__r_e_a_d_m_e_autotoc_md594}{}\doxysection{Installation}\label{md_renatio_todos__r_e_a_d_m_e_autotoc_md594}
There are couple ways to install this plugin.


\begin{DoxyEnumerate}
\item Use October \href{http://octobercms.com/help/site/marketplace}{\texttt{ Marketplace}} and {\bfseries{Add to project}} button.
\item Use October backend area {\itshape Settings $>$ System $>$ Updates \& Plugins $>$ Install Plugins} and type {\bfseries{Renatio.\+Todos}}.
\item Use {\ttfamily php artisan plugin\+:install Renatio.\+Todos} command.
\item Use {\ttfamily composer require renatio/todos-\/plugin} in project root.
\end{DoxyEnumerate}\hypertarget{md_renatio_todos__r_e_a_d_m_e_autotoc_md595}{}\doxysection{Configuration}\label{md_renatio_todos__r_e_a_d_m_e_autotoc_md595}
Plugin relies on the system scheduler for sending reminder email. For scheduled tasks to operate correctly, you should configure \href{http://octobercms.com/docs/help/installation\#crontab-setup}{\texttt{ Cron entry on your server}}.\hypertarget{md_renatio_todos__r_e_a_d_m_e_autotoc_md596}{}\doxysection{Using}\label{md_renatio_todos__r_e_a_d_m_e_autotoc_md596}
Plugin will register menu item called {\bfseries{To-\/\+Do List}}, which allow you to manage lists and tasks.

For each task you can set reminder at column. This will send reminder email to user at given time.

User can see only lists that belongs to him.\hypertarget{md_renatio_todos__r_e_a_d_m_e_autotoc_md597}{}\doxysection{Mail template}\label{md_renatio_todos__r_e_a_d_m_e_autotoc_md597}
Plugin will register reminder email template. You can customize it in {\itshape Settings $>$ Mail Templates}.\hypertarget{md_renatio_todos__r_e_a_d_m_e_autotoc_md598}{}\doxysection{Support}\label{md_renatio_todos__r_e_a_d_m_e_autotoc_md598}
