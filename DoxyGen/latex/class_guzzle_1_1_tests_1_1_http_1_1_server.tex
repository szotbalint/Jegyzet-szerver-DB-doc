\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_guzzle_1_1_tests_1_1_http_1_1_server}{Server}} class is used to control a scripted webserver using node.\+js that will respond to HTTP requests with queued responses.

Queued responses will be served to requests using a FIFO order. All requests received by the server are stored on the node.\+js server and can be retrieved by calling \{\begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{class_guzzle_1_1_tests_1_1_http_1_1_server_a64c3ca8ddd1f058c266ce466e8e26326}{Server\+::get\+Received\+Requests()}}\}.
\end{DoxySeeAlso}
\mbox{\hyperlink{namespace_guzzle_1_1_tests_1_1_mock}{Mock}} responses that don\textquotesingle{}t require data to be transmitted over HTTP a great for testing. \mbox{\hyperlink{namespace_guzzle_1_1_tests_1_1_mock}{Mock}} response, however, cannot test the actual sending of an HTTP request using c\+URL. This test server allows the simulation of any number of HTTP request response transactions to test the actual sending of requests over the wire without having to leave an internal network.
