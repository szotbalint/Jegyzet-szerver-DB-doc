\doxysubsection{Detailed Description}
Abstract Response

This abstract class implements \mbox{\hyperlink{interface_omnipay_1_1_common_1_1_message_1_1_response_interface}{Response\+Interface}} and defines a basic set of functions that all \mbox{\hyperlink{class_omnipay_1_1_omnipay}{Omnipay}} Requests are intended to include.

Objects of this class or a subclass are usually created in the Request object (subclass of \mbox{\hyperlink{class_omnipay_1_1_common_1_1_message_1_1_abstract_request}{Abstract\+Request}}) as the return parameters from the send() function.

Example -- validating and sending a request\+:

{\ttfamily  \$my\+Response = \$my\+Request-\/$>$send(); // now do something with the \$my\+Response object, test for success, etc. }
