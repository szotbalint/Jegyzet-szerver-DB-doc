\doxysubsection{Detailed Description}
Visitor used to apply a body to a request

This visitor can use a data parameter of \textquotesingle{}expect\textquotesingle{} to control the Expect header. Set the expect data parameter to false to disable the expect header, or set the value to an integer so that the expect 100-\/continue header is only added if the Content-\/\+Length of the entity body is greater than the value.
