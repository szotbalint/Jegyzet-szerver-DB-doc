\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_omnipay_1_1_omnipay}{Omnipay}} class

Provides static access to the gateway factory methods. This is the recommended route for creation and establishment of payment gateway objects via the standard Gateway\+Factory.

Example\+:

{\ttfamily  // Create a gateway for the \mbox{\hyperlink{namespace_omnipay_1_1_pay_pal}{Pay\+Pal}} Express\+Gateway // (routes to \mbox{\hyperlink{class_omnipay_1_1_common_1_1_gateway_factory_add7982ae8c25f1e10b44271426feac99}{Gateway\+Factory\+::create}}) \$gateway = Omnipay\+::create(\textquotesingle{}Express\+Gateway\textquotesingle{});}

{\ttfamily  // Initialise the gateway \$gateway-\/$>$initialize(...);}

{\ttfamily  // Get the gateway parameters. \$parameters = \$gateway-\/$>$get\+Parameters();}

{\ttfamily  // Create a credit card object \$card = new Credit\+Card(...);}

{\ttfamily  // Do an authorisation transaction on the gateway if (\$gateway-\/$>$supports\+Authorize()) \{ \$gateway-\/$>$authorize(...); \} else \{ throw new \textbackslash{}\+Exception(\textquotesingle{}Gateway does not support authorize()\textquotesingle{}); \} }

For further code examples see the {\itshape omnipay-\/example} repository on github.

Static function call router.
