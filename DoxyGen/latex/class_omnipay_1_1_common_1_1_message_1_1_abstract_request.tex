\doxysubsection{Detailed Description}
Abstract Request

This abstract class implements \mbox{\hyperlink{interface_omnipay_1_1_common_1_1_message_1_1_request_interface}{Request\+Interface}} and defines a basic set of functions that all \mbox{\hyperlink{class_omnipay_1_1_omnipay}{Omnipay}} Requests are intended to include.

Requests of this class are usually created using the create\+Request function of the gateway and then actioned using methods within this class or a class that extends this class.

Example -- creating a request\+:

{\ttfamily  class My\+Request extends \textbackslash{}\+Omnipay\textbackslash{}\+Common\textbackslash{}\+Message\textbackslash{}\+Abstract\+Request \{\};}

{\ttfamily  class My\+Gateway extends \textbackslash{}\+Omnipay\textbackslash{}\+Common\textbackslash{}\+Abstract\+Gateway \{ function my\+Request(\$parameters) \{ \$this-\/$>$create\+Request(\textquotesingle{}My\+Request\textquotesingle{}, \$parameters); \} \}}

{\ttfamily  // Create the gateway object \$gw = Omnipay\+::create(\textquotesingle{}My\+Gateway\textquotesingle{});}

{\ttfamily  // Create the request object \$my\+Request = \$gw-\/$>$my\+Request(\$some\+Parameters); }

Example -- validating and sending a request\+:

{\ttfamily  try \{ \$my\+Request-\/$>$\mbox{\hyperlink{class_omnipay_1_1_common_1_1_message_1_1_abstract_request_a184909dab34698899937d810a9f5d393}{validate()}}; \$my\+Response = \$my\+Request-\/$>$\mbox{\hyperlink{class_omnipay_1_1_common_1_1_message_1_1_abstract_request_a12bcef5130168b80d3d52dc82213f19a}{send()}}; \} catch (Invalid\+Request\+Exception \$e) \{ print \char`\"{}\+Something went wrong\+: \char`\"{} . \$e-\/$>$get\+Message() . \char`\"{}\textbackslash{}n\char`\"{}; \} // now do something with the \$my\+Response object, test for success, etc. }
