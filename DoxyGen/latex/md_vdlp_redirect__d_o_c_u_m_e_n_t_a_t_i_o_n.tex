This plugin should be easy to understand if you are familiar with the basics of the web. If you have issues setting up some redirects, please do not hesitate to contact me.\hypertarget{md_vdlp_redirect__d_o_c_u_m_e_n_t_a_t_i_o_n_autotoc_md705}{}\doxysection{Redirect types}\label{md_vdlp_redirect__d_o_c_u_m_e_n_t_a_t_i_o_n_autotoc_md705}
This plugins ships with two types of redirects\+:


\begin{DoxyItemize}
\item {\bfseries{Exact}}; performs an exact match on the Source path.
\item {\bfseries{Placeholders}}; matches placeholders like \{id\} or \{category\} (like the defined routes in Symfony or Laravel framework).
\item {\bfseries{Regular expression}}; Use regular expressions to match multiple patterns at once (advanced).
\end{DoxyItemize}\hypertarget{md_vdlp_redirect__d_o_c_u_m_e_n_t_a_t_i_o_n_autotoc_md706}{}\doxysection{Redirect target types}\label{md_vdlp_redirect__d_o_c_u_m_e_n_t_a_t_i_o_n_autotoc_md706}
This plugin allows you to redirect to the following types\+:


\begin{DoxyItemize}
\item An internal path
\item An internal CMS Page
\item An internal Static Page ({\ttfamily Rain\+Lab.\+Pages} plugin)
\item An external URL
\end{DoxyItemize}\hypertarget{md_vdlp_redirect__d_o_c_u_m_e_n_t_a_t_i_o_n_autotoc_md707}{}\doxysection{Relative vs. Absolute URLs}\label{md_vdlp_redirect__d_o_c_u_m_e_n_t_a_t_i_o_n_autotoc_md707}
Both types are supported by the Redirect plugin. Absolute URLs are generated by default. This setting can be changed at the Redirect Settings page.

In some cases it is necessary to use Relative URLs only. When using a reverse proxy for example. Or when you are hosting multiple domains on one single codebase.

Example when Absolute URLs are enabled (default)\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{Source path: /path/from}
\DoxyCodeLine{Target path: /path/to}
\DoxyCodeLine{Result path when matched: https://example.com/path/to}

\end{DoxyCode}


Example when Relative URLs are enabled\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{Source path: /path/from}
\DoxyCodeLine{Target path: /path/to}
\DoxyCodeLine{Result path when matched: /path/to}

\end{DoxyCode}
\hypertarget{md_vdlp_redirect__d_o_c_u_m_e_n_t_a_t_i_o_n_autotoc_md708}{}\doxysection{Scheme matching}\label{md_vdlp_redirect__d_o_c_u_m_e_n_t_a_t_i_o_n_autotoc_md708}
This plugin allows you to match requests from a {\ttfamily \href{http://}{\texttt{ http\+://}}} scheme to a {\ttfamily \href{https://}{\texttt{ https\+://}}} scheme and vice versa.\hypertarget{md_vdlp_redirect__d_o_c_u_m_e_n_t_a_t_i_o_n_autotoc_md709}{}\doxysection{Placeholders}\label{md_vdlp_redirect__d_o_c_u_m_e_n_t_a_t_i_o_n_autotoc_md709}
Every placeholder can be attached to a requirement. A requirement consists of a {\ttfamily placeholder}, {\ttfamily requirement} and an optional {\ttfamily replacement} value.

Example\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{Input path:}
\DoxyCodeLine{/blog.php?category=cat\&id=145}
\DoxyCodeLine{}
\DoxyCodeLine{Source path:}
\DoxyCodeLine{/blog.php?category=\{category\}\&id=\{id\}}
\DoxyCodeLine{}
\DoxyCodeLine{Target path:}
\DoxyCodeLine{/blog/\{category\}/\{id\}}
\DoxyCodeLine{}
\DoxyCodeLine{Result path:}
\DoxyCodeLine{/blog/cat/145}

\end{DoxyCode}



\begin{DoxyItemize}
\item The requirement for {\ttfamily \{category\}} would be\+: {\ttfamily \mbox{[}a-\/z\+A-\/Z\mbox{]}} or could be more specific like {\ttfamily (dog$\vert$cat$\vert$mouse)}.
\item The requirement for {\ttfamily \{id\}} would be\+: {\ttfamily \mbox{[}0-\/9\mbox{]}+}.
\end{DoxyItemize}

{\bfseries{Replacement value}}

A requirement can also contain a replacement value. Provide this replacement value if you need to rewrite a certain placeholder to a static value.

Example\+:

The requirement for {\ttfamily \{category\}} is {\ttfamily (dog$\vert$cat$\vert$mouse)}, with replacement value {\ttfamily animals}.


\begin{DoxyCode}{0}
\DoxyCodeLine{Input path:}
\DoxyCodeLine{/blog.php?category=mouse\&id=1337}
\DoxyCodeLine{}
\DoxyCodeLine{Source path:}
\DoxyCodeLine{/blog.php?category=\{category\}\&id=\{id\}}
\DoxyCodeLine{}
\DoxyCodeLine{Target path:}
\DoxyCodeLine{/blog/\{category\}/\{id\}}
\DoxyCodeLine{}
\DoxyCodeLine{Result:}
\DoxyCodeLine{/blog/animals/1337}

\end{DoxyCode}
