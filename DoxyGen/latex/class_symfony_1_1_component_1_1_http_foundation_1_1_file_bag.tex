Fixes a malformed PHP \$\+\_\+\+FILES array.

PHP has a bug that the format of the \$\+\_\+\+FILES array differs, depending on whether the uploaded file fields had normal field names or array-\/like field names (\char`\"{}normal\char`\"{} vs. \char`\"{}parent\mbox{[}child\mbox{]}\char`\"{}).

This method fixes the array to look like the \char`\"{}normal\char`\"{} \$\+\_\+\+FILES array.

It\textquotesingle{}s safe to pass an already converted array, in which case this method just returns the original array unmodified.
