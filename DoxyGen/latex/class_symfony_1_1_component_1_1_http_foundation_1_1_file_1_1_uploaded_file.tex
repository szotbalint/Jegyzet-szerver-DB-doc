Accepts the information of the uploaded file as provided by the PHP global \$\+\_\+\+FILES.

The file object is only created when the uploaded file is valid (i.\+e. when the \mbox{\hyperlink{class_symfony_1_1_component_1_1_http_foundation_1_1_file_1_1_uploaded_file_a7b37efab7473a1effc29f8be2421f6e3}{is\+Valid()}} method returns true). Otherwise the only methods that could be called on an \mbox{\hyperlink{class_symfony_1_1_component_1_1_http_foundation_1_1_file_1_1_uploaded_file}{Uploaded\+File}} instance are\+:


\begin{DoxyItemize}
\item get\+Client\+Original\+Name,
\item get\+Client\+Mime\+Type,
\item is\+Valid,
\item get\+Error.
\end{DoxyItemize}

Calling any other method on an non-\/valid instance will cause an unpredictable result.
