\doxysubsection{Detailed Description}
\mbox{\hyperlink{namespace_omnipay_1_1_pay_pal}{Pay\+Pal}} REST Complete Purchase Request

Use this message to execute (complete) a \mbox{\hyperlink{namespace_omnipay_1_1_pay_pal}{Pay\+Pal}} payment that has been approved by the payer. You can optionally update transaction information when executing the payment by passing in one or more transactions.

This call only works after a buyer has approved the payment using the provided \mbox{\hyperlink{namespace_omnipay_1_1_pay_pal}{Pay\+Pal}} approval URL.\hypertarget{class_omnipay_1_1_pay_pal_1_1_message_1_1_rest_complete_purchase_request_autotoc_md149}{}\doxyparagraph{Example}\label{class_omnipay_1_1_pay_pal_1_1_message_1_1_rest_complete_purchase_request_autotoc_md149}
The payer ID and the payment ID returned from the callback after the purchase will be passed to the return URL as GET parameters payer\+Id and payment\+Id respectively.

See \mbox{\hyperlink{class_omnipay_1_1_pay_pal_1_1_message_1_1_rest_purchase_request}{Rest\+Purchase\+Request}} for the first part of this example transaction\+:
