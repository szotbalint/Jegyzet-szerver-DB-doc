\doxysubsection{Detailed Description}
\mbox{\hyperlink{namespace_omnipay_1_1_pay_pal}{Pay\+Pal}} REST Create Subscription Request

Use this call to create a billing agreement for the buyer. The response for this call includes these HATEOAS links\+: an approval\+\_\+url link and an execute link. Each returned link includes the token for the agreement.

For \mbox{\hyperlink{namespace_omnipay_1_1_pay_pal}{Pay\+Pal}} payments\+:


\begin{DoxyItemize}
\item After successfully creating the agreement, direct the user to the approval\+\_\+url on the \mbox{\hyperlink{namespace_omnipay_1_1_pay_pal}{Pay\+Pal}} site so that the user can approve the agreement.
\item Call the execute link to execute the billing agreement.
\end{DoxyItemize}

Note\+: Billing agreements for credit card payments execute automatically when created. There is no need for the user to approve the agreement or to execute the agreement.\hypertarget{class_omnipay_1_1_pay_pal_1_1_message_1_1_rest_create_subscription_request_autotoc_md158}{}\doxyparagraph{Request Data}\label{class_omnipay_1_1_pay_pal_1_1_message_1_1_rest_create_subscription_request_autotoc_md158}
Pass the agreement details in the body of a POST call, including the following agreement object properties\+:


\begin{DoxyItemize}
\item name (string). Required.
\item description (string). Required.
\item start\+\_\+date (string). Format yyyy-\/\+MM-\/dd z, as defined in ISO8601. Required.
\item agreement\+\_\+details (array)
\item payer (array). Required
\item shipping\+\_\+address (array). Should be provided if it is different to the default address.
\item override\+\_\+merchant\+\_\+preferences (array).
\item override\+\_\+charge\+\_\+models (array).
\item plan (array). Required.
\end{DoxyItemize}\hypertarget{class_omnipay_1_1_pay_pal_1_1_message_1_1_rest_create_subscription_request_autotoc_md159}{}\doxyparagraph{Example}\label{class_omnipay_1_1_pay_pal_1_1_message_1_1_rest_create_subscription_request_autotoc_md159}
