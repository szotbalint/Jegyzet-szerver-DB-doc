\doxysubsection{Detailed Description}
Tester for checking if the response HTTP code is equal to the matched redirect.

Situations\+: a) Failing when given path matches a redirect but response code is not equal to response code. b) Failing when given path does not match but status code is not 301, 302, ... c) Passes when given path does not match with a redirect.
