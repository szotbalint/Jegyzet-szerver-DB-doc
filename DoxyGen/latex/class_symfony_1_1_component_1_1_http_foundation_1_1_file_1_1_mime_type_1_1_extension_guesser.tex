\doxysubsection{Detailed Description}
A singleton mime type to file extension guesser.

A default guesser is provided. You can register custom guessers by calling the \mbox{\hyperlink{class_symfony_1_1_component_1_1_http_foundation_1_1_file_1_1_mime_type_1_1_extension_guesser_a53b97d20de2aa57b200384d003087b8f}{register()}} method on the singleton instance\+: \begin{DoxyVerb}$guesser = ExtensionGuesser::getInstance();
$guesser->register(new MyCustomExtensionGuesser());
\end{DoxyVerb}
 The last registered guesser is preferred over previously registered ones.

{\footnotesize\ttfamily guess (\begin{DoxyParamCaption}\item[{}]{\$mime\+Type }\end{DoxyParamCaption})}

Tries to guess the extension.

The mime type is passed to each registered mime type guesser in reverse order of their registration (last registered is queried first). Once a guesser returns a value that is not NULL, this method terminates and returns the value.
