\doxysubsection{Detailed Description}
An Event\+Subscriber knows himself what events he is interested in. If an Event\+Subscriber is added to an \mbox{\hyperlink{interface_symfony_1_1_component_1_1_event_dispatcher_1_1_event_dispatcher_interface}{Event\+Dispatcher\+Interface}}, the manager invokes \mbox{\hyperlink{interface_symfony_1_1_component_1_1_event_dispatcher_1_1_event_subscriber_interface_af1f63475927bbcb3bd21b58712076104}{get\+Subscribed\+Events}} and registers the subscriber as a listener for all returned events.

{\footnotesize\ttfamily static get\+Subscribed\+Events (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [static]}}

Returns an array of event names this subscriber wants to listen to.

The array keys are event names and the value can be\+:


\begin{DoxyItemize}
\item The method name to call (priority defaults to 0)
\item An array composed of the method name to call and the priority
\item An array of arrays composed of the method names to call and respective priorities, or 0 if unset
\end{DoxyItemize}

For instance\+:


\begin{DoxyItemize}
\item array(\textquotesingle{}event\+Name\textquotesingle{} =$>$ \textquotesingle{}method\+Name\textquotesingle{})
\item array(\textquotesingle{}event\+Name\textquotesingle{} =$>$ array(\textquotesingle{}method\+Name\textquotesingle{}, \$priority))
\item array(\textquotesingle{}event\+Name\textquotesingle{} =$>$ array(array(\textquotesingle{}method\+Name1\textquotesingle{}, \$priority), array(\textquotesingle{}method\+Name2\textquotesingle{})))
\end{DoxyItemize}
