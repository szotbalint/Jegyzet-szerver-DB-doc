\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_guzzle_1_1_service_1_1_command_1_1_closure_command}{Closure\+Command}} is a command that allows dynamic commands to be created at runtime using a closure to prepare the request. A closure key and \textbackslash{}\+Closure value must be passed to the command in the constructor. The closure must accept the command object as an argument.
