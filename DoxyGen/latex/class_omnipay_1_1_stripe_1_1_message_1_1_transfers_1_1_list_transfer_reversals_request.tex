\doxysubsection{Detailed Description}
\mbox{\hyperlink{namespace_omnipay_1_1_stripe}{Stripe}} List Transfer Reversals Request.

You can see a list of the reversals belonging to a specific transfer.

Note that the 10 most recent reversals are always available by default on the transfer object. If you need more than those 10, you can use this API method and the {\ttfamily limit} and {\ttfamily starting\+\_\+after} parameters to page through additional reversals.
