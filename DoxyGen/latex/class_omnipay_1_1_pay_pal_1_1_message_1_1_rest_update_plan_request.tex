\mbox{\hyperlink{namespace_omnipay_1_1_pay_pal}{Pay\+Pal}} REST Update Plan Request

You can update the information for an existing billing plan. The state of a plan must be active before a billing agreement is created.\hypertarget{class_omnipay_1_1_pay_pal_1_1_message_1_1_rest_update_plan_request_autotoc_md188}{}\doxyparagraph{Request Data}\label{class_omnipay_1_1_pay_pal_1_1_message_1_1_rest_update_plan_request_autotoc_md188}
