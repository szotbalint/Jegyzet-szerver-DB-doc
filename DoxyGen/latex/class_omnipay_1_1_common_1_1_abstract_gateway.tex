\doxysubsection{Detailed Description}
Base payment gateway class

This abstract class should be extended by all payment gateways throughout the \mbox{\hyperlink{class_omnipay_1_1_omnipay}{Omnipay}} system. It enforces implementation of the \mbox{\hyperlink{interface_omnipay_1_1_common_1_1_gateway_interface}{Gateway\+Interface}} interface and defines various common attibutes and methods that all gateways should have.

Example\+:

{\ttfamily  // Initialise the gateway \$gateway-\/$>$initialize(...);}

{\ttfamily  // Get the gateway parameters. \$parameters = \$gateway-\/$>$\mbox{\hyperlink{class_omnipay_1_1_common_1_1_abstract_gateway_a015cb52e5774a1972d296c9694d2a3c3}{get\+Parameters()}};}

{\ttfamily  // Create a credit card object \$card = new Credit\+Card(...);}

{\ttfamily  // Do an authorisation transaction on the gateway if (\$gateway-\/$>$\mbox{\hyperlink{class_omnipay_1_1_common_1_1_abstract_gateway_af0f3b42fd5a08c7b2b9ca5f575ec05d7}{supports\+Authorize()}}) \{ \$gateway-\/$>$authorize(...); \} else \{ throw new \textbackslash{}\+Exception(\textquotesingle{}Gateway does not support authorize()\textquotesingle{}); \} }

For further code examples see the {\itshape omnipay-\/example} repository on github.
