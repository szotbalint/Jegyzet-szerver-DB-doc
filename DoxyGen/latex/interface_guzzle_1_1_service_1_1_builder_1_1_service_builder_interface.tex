\doxysubsection{Detailed Description}
\mbox{\hyperlink{namespace_guzzle_1_1_service}{Service}} builder used to store and build clients or arbitrary data. \mbox{\hyperlink{class_guzzle_1_1_service_1_1_client}{Client}} configuration data can be supplied to tell the service builder how to create and cache \{\begin{DoxySeeAlso}{See also}
\textbackslash{}\+Guzzle\textbackslash{}\+Service\textbackslash{}\+Client\+Interface\} objects. Arbitrary data can be supplied and accessed from a service builder. Arbitrary data and other clients can be referenced by name in client configuration arrays to make them input for building other clients (e.\+g. \char`\"{}\{key\}\char`\"{}).
