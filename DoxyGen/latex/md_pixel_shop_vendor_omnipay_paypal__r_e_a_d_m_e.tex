\begin{DoxyItemize}
\item Pay\+Pal\+\_\+\+Express (Pay\+Pal Express Checkout)
\item Pay\+Pal\+\_\+\+Express\+In\+Context (Pay\+Pal Express In-\/\+Context Checkout)

\item Pay\+Pal\+\_\+\+Rest (Paypal Rest API)

For general usage instructions, please see the main \href{https://github.com/thephpleague/omnipay}{\texttt{ Omnipay}} repository.\hypertarget{md_pixel_shop_vendor_omnipay_paypal__r_e_a_d_m_e_autotoc_md137}{}\doxysection{Quirks}\label{md_pixel_shop_vendor_omnipay_paypal__r_e_a_d_m_e_autotoc_md137}
The transaction reference obtained from the purchase() response can\textquotesingle{}t be used to refund a purchase. The transaction reference from the complete\+Purchase() response is the one that should be used.\hypertarget{md_pixel_shop_vendor_omnipay_paypal__r_e_a_d_m_e_autotoc_md138}{}\doxysection{Out Of Scope}\label{md_pixel_shop_vendor_omnipay_paypal__r_e_a_d_m_e_autotoc_md138}
Omnipay does not cover recurring payments or billing agreements, and so those features are not included in this package. Extensions to this gateway are always welcome.\hypertarget{md_pixel_shop_vendor_omnipay_paypal__r_e_a_d_m_e_autotoc_md139}{}\doxysection{Support}\label{md_pixel_shop_vendor_omnipay_paypal__r_e_a_d_m_e_autotoc_md139}
