\doxysubsection{Detailed Description}
Interface used for loading configuration data (service descriptions, service builder configs, etc)

If a loaded configuration data sets includes a top level key containing an \textquotesingle{}includes\textquotesingle{} section, then the data in the file will extend the merged result of all of the included config files.
