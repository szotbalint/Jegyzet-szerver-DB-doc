\doxysubsection{Detailed Description}
\mbox{\hyperlink{namespace_omnipay_1_1_pay_pal}{Pay\+Pal}} REST Search Transaction Request

Use this call to search for the transactions within a billing agreement. Note that this is not a generic transaction search function -- for that see \mbox{\hyperlink{class_omnipay_1_1_pay_pal_1_1_message_1_1_rest_list_purchase_request}{Rest\+List\+Purchase\+Request}}. It only searches for transactions within a billing agreement.

This should be used on a regular basis to determine the success / failure state of transactions on active billing agreements.\hypertarget{class_omnipay_1_1_pay_pal_1_1_message_1_1_rest_search_transaction_request_autotoc_md181}{}\doxyparagraph{Example}\label{class_omnipay_1_1_pay_pal_1_1_message_1_1_rest_search_transaction_request_autotoc_md181}
