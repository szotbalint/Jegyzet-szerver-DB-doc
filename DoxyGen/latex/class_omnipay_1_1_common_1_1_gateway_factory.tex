\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_omnipay_1_1_omnipay}{Omnipay}} Gateway Factory class

This class abstracts a set of gateways that can be independently registered, accessed, and used.

Note that static calls to the \mbox{\hyperlink{class_omnipay_1_1_omnipay}{Omnipay}} class are routed to this class by the static call router (\+\_\+\+\_\+call\+Static) in \mbox{\hyperlink{class_omnipay_1_1_omnipay}{Omnipay}}.

Example\+:

{\ttfamily  // Create a gateway for the \mbox{\hyperlink{namespace_omnipay_1_1_pay_pal}{Pay\+Pal}} Express\+Gateway // (routes to \mbox{\hyperlink{class_omnipay_1_1_common_1_1_gateway_factory_add7982ae8c25f1e10b44271426feac99}{Gateway\+Factory\+::create}}) \$gateway = Omnipay\+::create(\textquotesingle{}Express\+Gateway\textquotesingle{}); }
